Данная программа реализует многослойный персептрон для решения задачи классификации
по логической функции XOR (исключающее ИЛИ).

**Цель программы** — показать, как нейронная сеть может решать нелинейно разделимую задачу
с помощью скрытого слоя и нелинейных функций активации.

**1. Генерация обучающих данных**

* Создаётся четыре группы точек, расположенных по вершинам квадрата:
(0,0), (0,1), (1,0), (1,1).

* Для наглядности к координатам добавляется небольшой случайный шум, чтобы точки образовывали четыре компактных облака.

Каждое облако относится к своему классу в соответствии с логикой XOR:

* Класс 0: точки (0,0) и (1,1)

* Класс 1: точки (0,1) и (1,0)

**2️. Построение и обучение нейронной сети**

Используется модель MLPClassifier из библиотеки scikit-learn.

Архитектура сети:

* 2 входных нейрона (x₁ и x₂);

* 1 скрытый слой из 2 нейронов (формируют внутренние признаки);

* 1 выходной нейрон (результат функции XOR).

В качестве функции активации применяется tanh, что позволяет модели строить нелинейные границы решений.

Обучение выполняется методом обратного распространения ошибки (backpropagation).

**3️. Проверка логики XOR**

* После обучения программа проверяет четыре основные комбинации входов:
(0,0), (0,1), (1,0), (1,1)

* Результат классификации подтверждает корректную реализацию логической функции XOR.

**4️. Визуализация результатов**

Программа строит несколько графиков, иллюстрирующих внутреннюю логику работы сети:

* Первый график: активация первого скрытого нейрона (поведение похоже на функцию OR).

* Второй график: активация второго нейрона (поведение похоже на NAND).

* Третий график: итоговое разделение классов (функция XOR).

* Четвёртый график: крупное изображение диагонального разделения классов с обозначением областей классов “0” и “1”.


Многослойный персептрон не просто подбирает веса, а учится формировать внутренние признаки, позволяющие линейно разделить исходно нелинейную задачу.

В случае функции XOR:

* Первый слой формирует логические признаки, аналогичные OR и NAND;

* Второй слой объединяет их и реализует XOR.


**После запуска программа:**

* отображает на экране графики с визуализацией внутренней логики сети;

* демонстрирует корректную работу функции XOR
(1, если входы различаются; 0, если совпадают);

* наглядно показывает, как многослойный персептрон делит точки по диагонали, реализуя логику исключающего ИЛИ.
